# 바이브 랜딩페이지

AI 기반 부동산 분석 플랫폼 "바이브"의 GitHub Pages 랜딩페이지입니다.

## 프로젝트 소개

바이브는 **아파트 실거래가·트렌드·예측가격·단지 인사이트**를 하나의 화면에서 제공하고, 거래 변화 시점을 자동으로 알려주는 **타이밍 기반 의사결정 솔루션**입니다.

### 핵심 가치 제안

"언제 사고 팔아야 할지 판단하기 어려운" 사용자들이 기존의 단순 실거래 조회 앱보다 더 빠르고 정확하게, 실거래 추세·예측가격·변화 시점 인사이트를 한 화면에서 확인해 타이밍 의사결정을 명확하게 하도록 돕습니다.

### 핵심 기능 (MVP)

- 📈 **실거래가 히스토리 차트**: 단지별 가격 흐름을 직관적으로 시각화 (1/3/6/12개월 필터)
- 🤖 **AI 가격 예측**: OpenAI 기반 1개월, 3개월 예측 및 상승/하락 확률 제공
- 💡 **단지 인사이트 자동 생성**: AI 기반 자동 분석 및 타이밍 의견
- 🔔 **변화 감지 알림**: 거래량 급증, 가격 급등/급락 등 중요한 변화를 실시간으로 감지
- ⚖️ **단지·평형 비교**: 여러 단지를 나란히 비교하여 최적의 선택 지원
- ⭐ **관심 단지 관리**: 관심 단지 최대 20개 저장 및 알림 설정

## 배포

이 페이지는 GitHub Pages를 통해 자동으로 배포됩니다. `main` 브랜치에 푸시할 때마다 자동으로 배포됩니다.

GitHub Actions 워크플로가 자동으로 배포를 처리합니다:
- `main` 브랜치에 푸시 시 자동 배포
- GitHub Pages 환경에서 자동 호스팅

## 로컬에서 실행

1. 파일을 다운로드합니다
2. `index.html` 파일을 브라우저에서 열거나 로컬 서버를 실행합니다:

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server
```

그 후 브라우저에서 `http://localhost:8000`으로 접속하세요.

## 파일 구조

```
landing-page/
├── index.html              # 메인 랜딩 페이지
├── css/
│   └── style.css          # 스타일시트
├── js/
│   └── main.js            # 인터랙션 스크립트
├── .github/
│   └── workflows/
│       └── deploy.yml     # GitHub Actions 워크플로
├── plans/                 # 계획 문서
└── README.md              # 이 파일
```

## 기술 스택

### 랜딩페이지
- HTML5 (시맨틱 마크업)
- CSS3 (CSS Variables, Flexbox, Grid, 반응형 디자인)
- JavaScript (바닐라 JS, Intersection Observer API)

### 배포
- GitHub Pages
- GitHub Actions (자동 배포)

### 프로젝트 본체 (참고)
- **Frontend**: Next.js
- **Backend**: Next.js API Routes
- **Database/Auth**: Supabase
- **AI**: OpenAI

## 타겟 고객

1. **예비 실거주자 (30~40대)**: 내 집 마련 준비 중, 단지별 흐름과 타이밍에 민감
2. **개인 아파트 투자자**: 단지 비교 및 변동 트렌드 파악 니즈
3. **중개업자**: 상담 시 최신 데이터와 인사이트 필요
