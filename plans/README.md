# GitHub 페이지 계획 문서

이 폴더에는 **실거래가 + 예측가격 + 인사이트 플랫폼** 프로젝트 소개를 위한 GitHub 페이지를 만들기 위한 계획 문서들이 포함되어 있습니다.

## 📋 문서 목록

### 1. [01-overview.md](./01-overview.md)
프로젝트 개요, 목표, 핵심 가치 제안, 문제 정의, 타겟 고객, 핵심 기능, 기술 스택 등 기본 정보

### 2. [02-structure.md](./02-structure.md)
GitHub 페이지의 전체 구조 및 파일 구조 계획

- 네비게이션 구조
- 파일 구조

### 3. [03-content.md](./03-content.md)
각 섹션별 콘텐츠 계획 및 작성 내용
- 히어로 섹션
- 문제 정의 섹션
- 해결책/기능 소개 섹션 (PRD 기반 상세)
- 사용 사례 섹션
- 기술 스택 섹션
- 로드맵 섹션
- CTA 섹션

### 4. [04-design.md](./04-design.md)
디자인 가이드라인
- 컬러 팔레트

### 5. [05-implementation.md](./05-implementation.md)
구현 계획

## 🎯 프로젝트 핵심 가치

**"언제 사고 팔아야 할지 판단하기 어려운" 사용자들이 기존의 단순 실거래 조회 앱보다 더 빠르고 정확하게, 실거래 추세·예측가격·변화 시점 인사이트를 한 화면에서 확인해 타이밍 의사결정을 명확하게 하도록 돕는다.**

- 명확한 문제 정의 및 해결책
