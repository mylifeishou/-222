# 디자인 가이드라인

## 디자인 컨셉
**현대적이고 전문적이며 신뢰감 있는 디자인**

## 컬러 팔레트

### 메인 컬러
- **Primary**: 부동산 관련 신뢰감 있는 색상 
  - 추천: #1565c0 (진한 파란색 - 신뢰감)
  - 또는: #1a237e (네이비 - 전문성)
- **Secondary**: 보조 색상
  - 추천: #ff6f00 (오렌지 - 액션/에너지)
  - 또는: #e65100 (다크 오렌지)

### 중립 색상
- **Background**: #ffffff (화이트)
- **Text Primary**: #212121 (진한 회색)
- **Text Secondary**: #757575 (중간 회색)
- **Border**: #e0e0e0 (연한 회색)

### 액센트
- **Success/Positive**: #4caf50 (초록색 - 상승 트렌드)
- **Warning/Negative**: #f44336 (빨간색 - 하락 트렌드)
- **Info**: #2196f3 (파란색 - 정보)

## 타이포그래피

### 제목
- **H1**: 대형, 굵게, 명확한 계층 구조
- **H2**: 중형, 섹션 구분
- **H3**: 소형, 서브 섹션

### 본문
- 기본 폰트 크기: 16px
- 줄 간격: 1.6
- 폰트 패밀리: 시스템 폰트 스택 (San Francisco, Segoe UI, 등)

## 레이아웃

### 그리드 시스템
- 최대 너비: 1200px
- 컨테이너 패딩: 24px (모바일), 48px (데스크톱)
- 섹션 간격: 80px (데스크톱), 48px (모바일)

### 반응형
- **모바일**: 320px - 767px
- **태블릿**: 768px - 1023px
- **데스크톱**: 1024px 이상

## 컴포넌트 스타일

### 버튼
- Primary: 배경색 + 흰색 텍스트, 둥근 모서리 (8px)
- Secondary: 테두리 + 텍스트 색상
- 호버 효과: 약간의 그림자 및 색상 변화

### 카드
- 배경: 흰색
- 그림자: 부드러운 그림자 (box-shadow)
- 둥근 모서리: 12px
- 패딩: 24px

### 차트 스타일
- 명확한 색상 구분
- 호버 시 상세 정보 표시
- 반응형 크기 조절

## 이미지/아이콘

### 아이콘 스타일
- 단순하고 명확한 라인 아이콘
- 일관된 스타일 유지

### 스크린샷
- 실제 기능 캡처
- 프레임 처리 (선택 사항)
- 반응형 이미지 사용

## 애니메이션

### 마이크로 인터랙션
- 버튼 호버 효과 (0.2s)
- 스크롤 애니메이션 (fade-in)
- 부드러운 전환 효과

### 성능 고려
- CSS 애니메이션 우선 사용
- GPU 가속 활용
- 최적화된 애니메이션 시간
